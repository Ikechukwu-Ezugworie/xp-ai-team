# User Stories – Chat Application

| ID | Story | Acceptance Criteria | Points |
|----|-------|---------------------|--------|
| US01 | As a user I want to register with username/email/password | Registration succeeds; duplicate rejected | 3 |
| US02 | As a user I want to log in and receive a token | Token returned on valid credentials | 3 |
| US03 | As a user I want to log out | Session invalidated | 1 |
| US04 | As a user I want to see a list of chat rooms | Rooms displayed on login | 2 |
| US05 | As a user I want to create a new chat room | Room appears in list immediately | 3 |
| US06 | As a user I want to join an existing room | User added to room member list | 2 |
| US07 | As a user I want to send a text message in a room | Message appears for all members | 3 |
| US08 | As a user I want to see message history when I join | Last 50 messages loaded | 3 |
| US09 | As a user I want to see who else is in the room | Active member list shown | 2 |
| US10 | As a user I want real-time delivery of new messages | Message appears without refresh | 5 |
| US11 | As a user I want my username shown next to my messages | Username visible on each message | 1 |
| US12 | As a user I want timestamps on messages | Timestamp shown per message | 1 |
| US13 | As a user I want to leave a room | User removed from member list | 2 |
| US14 | As a user I want to see typing indicators | "User is typing…" shown in real time | 3 |
| US15 | As a user I want to search messages in a room | Search returns matching results | 5 |
| US16 | As an admin I want to delete a room | Room and messages removed | 3 |
| US17 | As a user I want mobile-responsive UI | Layout works on 375px viewport | 3 |
| US18 | As a user I want my session to persist on refresh | Token stored and reused | 2 |
